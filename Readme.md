# 📝 To-Do List Application

A simple To-Do List application built with **HTML, CSS, and JavaScript**. It allows users to manage daily tasks efficiently with Local Storage support.

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Save tasks using Local Storage
* Tasks persist after refreshing the browser

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📚 Concepts Practiced

* DOM Manipulation
* Event Handling
* Arrays and Objects
* Functions
* Local Storage
* JSON (`JSON.stringify()` and `JSON.parse()`)

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
